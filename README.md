A fun, gamified, and highly interactive typing tutor web application developed for **Sri Sathya Sai Vidya Vahini - Inclusive Education**. Designed specifically to help learners of all ages master keyboard typing through engaging visuals, progressive levels, and rewarding feedback.

---

## ✨ Key Features (Version 1 Release)

*   **Interactive Gameplay System:** 
    *   Real-time keyboard visualization.
    *   Animated hands guiding the user on correct finger placement (Light Mode).
*   **Profile Management & Onboarding:**
    *   Learners can create personalized profiles (Name, Age).
    *   Choose between interactive mascots: **Prem** (Boy) and **Shanti** (Girl).
    *   Switch seamlessly between multiple saved profiles.
*   **Progressive Level Design:**
    *   Unlockable levels with a rewarding visual sync (completed levels glow green).
    *   Dedicated **Practice Mode** for free-form typing exercises.
*   **Cinematic, Responsive UI:**
    *   Beautifully animated sky environments with floating clouds, twinkling stars, and waving mascots.
    *   Fully responsive across Desktop, Tablet, and Mobile.
    *   Built-in orientation detection prompting mobile users to use landscape mode for the best experience.
*   **Audio & Music:**
    *   Immersive background music and sound effects to keep learners engaged.

## 🚀 Getting Started

This project is built using strictly vanilla web technologies, requiring zero build steps or complex dependencies to run.

### Prerequisites
*   Any modern web browser (Google Chrome, Mozilla Firefox, Safari, Microsoft Edge).

### Installation & Usage
1. Clone or download the repository to your local machine.
2. Navigate to the project folder:
   ```bash
   cd type-tutor
   ```
3. Open `index.html` directly in your web browser.
   *(Alternatively, you can serve the directory using a lightweight local development server like VS Code Live Server or Python's `http.server`).*

---

## 🛠️ Tech Stack

*   **HTML5:** Semantic layout and accessibility standards.
*   **CSS3:** Extensive use of CSS variables, flexbox, grid, and high-performance keyframe animations (`transform` and `opacity`).
*   **JavaScript (ES6+):** Profile management, gameplay logic, bounding box calculations, and interactive state management.

## 📝 Release Notes (v1.0.0)

**Stable Production Release**
This version finalizes the core gameplay loops, profile system, and overall cinematic UI.

*   **Note on Dark Mode (Temporary V1 Fallback):** 
    To guarantee absolute layout stability for the V1 release, the animated typing hands have been intentionally hidden when **Dark Mode** is active. The keyboard, typing logic, and gameplay remain fully functional in both modes. Light mode includes the full hand-guided educational experience. A full CSS alignment fix for Dark Mode hands is planned for Version 2.
